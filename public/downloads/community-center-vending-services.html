Community Center Vending Pack

Multi-program planogram template, parks-and-rec procurement scope, weekend-event surge protocol

Community centers run multiple programs in parallel — youth sports, senior fitness, after-school programs, weekend events. Each program brings a different user demographic and demand curve. This pack codifies a multi-program planogram, the parks-and-recreation procurement scope, and a weekend-event surge protocol that keeps machines stocked during peak weekends.

3. Parks-and-recreation procurement scope

Parks-and-rec departments typically procure vending under one of three paths. Confirm path with department's purchasing officer.

Concession agreement (most common)
Operator pays commission (typically 8-25% of gross revenue) to the city / parks department. Term 3-5 years; renewable. No facility outlay. Concession agreement requires open bid in most municipalities.
Master contract / cooperative procurement
Some states maintain a master vending contract; cities / parks ride the master. Streamlined path. Examples: state-DOT master, regional COG cooperative.
Local RFP
Smaller parks departments issue their own RFP. Standard RFP language applies; pre-bid walkthrough recommended for multi-building scope.
Donation / sponsorship (limited)
Some health-focused programs (e.g., wellness foundation partnerships) fund subsidized hydration SKUs. Document gift-in-kind separately for accounting.

4. Weekend-event surge protocol

A weekend event (tournament, rental, family event) can do 3-5× a normal-day velocity. Without surge prep, machines stock-out by mid-day. The protocol:

  1. Event calendar shared with operator (4 weeks ahead)

    Facility shares confirmed weekend events 4 weeks in advance. Operator overlays against route schedule and adjusts pre-event restock.

  2. Pre-event restock (Friday)

    For events ≥ 100 expected attendees, operator restocks Friday close-of-business. Top-up Saturday morning if Saturday velocity tracks toward stockout.

  3. Surge planogram

    Hydration share elevated to 50% during high-attendance summer events. Replace lower-velocity SKUs temporarily with bulk-pack water + sports drink.

  4. Mid-event top-off

    Operator on-call schedule covers Saturday top-off for events ≥ 200 attendees. Threshold + SLA documented in contract.

  5. Post-event report

    Operator submits velocity + stockout report for events ≥ 100 attendees. Drives next-event surge plan.

5. Nutrition + healthy-share — community-center context

Parks-and-rec departments often align with public-health nutrition standards. Common references below.

USDA Smart Snacks (NSLP-adjacent)
School-day standard; some parks departments adopt for after-school programs. Sodium, sugar, fat caps per serving.
USDA Dietary Guidelines for Americans
Baseline for adult nutrition messaging. ≤ 200 mg sodium, ≤ 10 g added sugar typical thresholds.
CDC Healthier Food Retail standards
Public-sector framework for vending in government / community settings. Defines green / amber / red SKU classification.
Partnership for a Healthier America (PHA) framework
Adopted by many parks-and-rec departments. Operator certifies per-SKU PHA alignment.

6. ADA + accessibility (Title II)

Community centers are Title II facilities. Full ADA reach, clear-floor, operating-force requirements apply.

  • Reach range 15-48 in AFF; obstructed-reach exceptions documented (Section 308).
  • 30 in × 48 in clear floor space (Section 305).
  • Operating force ≤ 5 lbf; no tight grasping (Section 309.4).
  • Cashless option for users without coin/bill handling.
  • Tactile + high-contrast pricing where state code or facility policy requires.
  • Annual ADA recertification on file from operator.

7. Reporting + governance

Parks-and-rec director or facility manager reviews vending performance quarterly. The QBR pack:

  • Per-machine + per-building revenue, transaction count, average ticket.
  • Stockout incidents + resolution time.
  • Telemetry uptime % per machine.
  • Weekend-event report (velocity vs. forecast for each event ≥ 100 attendees).
  • Healthy-share % (count of SKUs meeting PHA / CDC / municipal criteria).
  • Concession commission to date vs. budget.
  • ADA + cleaning attestations current.
  • Worker / user-feedback channel summary.
