High School Vending Program Design Template

Compliance, placement, contract SLA, and after-school operations

High school vending sits at the intersection of USDA Smart Snacks (7 CFR 210.11), ADA Section 308, state competitive-food rules, and athletic-program revenue. This pack codifies the program design template so a high school is equipped to commit a full program — compliance, placement, contract SLA, after-school operations — rather than improvise.

1. Compliance baseline

High school vending has the broadest Smart Snacks scope of any K-12 segment because beverage rules are most permissive at HS (12oz milk/juice; 20oz calorie-free beverages allowed).

Smart Snacks — snacks
≤ 200 cal, ≤ 35% fat-calories, ≤ 10% saturated fat, ≤ 200 mg sodium, ≤ 35% sugar by weight.
Smart Snacks — entrée
≤ 350 cal, ≤ 35% fat-calories, ≤ 10% saturated fat, ≤ 480 mg sodium.
Smart Snacks — beverages
Plain water (any size); flavored water ≤ 5 cal/8oz; ≤ 12oz low-fat milk; ≤ 12oz 100% juice; ≤ 20oz calorie-free or ≤ 5 cal/8oz beverages.
Allergen labeling
FALCPA labels on every SKU; allergen summary on machine face.
FDA Food Code
Refrigerated units 35-40°F per 3-501.16; HACCP-style temp monitoring for fresh-food.
ADA Section 308
Operable parts ≤ 48 inches, ≥ 15 inches; ≤ 5 lbf operating force; clear floor space 30×48 inches.
State overlay
Confirm state competitive-food rules with the school nutrition director; some states (CA, MS, NY) tighten federal floor.

3. Contract SLA template

  1. Tier 1 — revenue-critical / cold-chain / payment failure

    Acknowledgement < 4 hours; resolution < 24 hours. Examples: machine offline, payment system down, refrigeration excursion.

  2. Tier 2 — stockout / planogram refresh

    Acknowledgement 24-48 hours; resolution 5-7 days.

  3. Tier 3 — cosmetic / signage

    Acknowledgement 5 days; resolution 14 days.

  4. Restock cadence

    Combo + beverage cooler: 2x/week baseline, scaling to 3-4x/week in peak windows (BTS, athletic seasons). Fresh-food: daily.

  5. Cashless availability

    Cashless uptime ≥ 99% per month; cash-and-cashless split reported monthly.

  6. Refund SLA

    Cashless refunds processed < 24 hours via app or SMS; cash refunds at next visit (max 7 days).

4. After-school operations playbook

Smart Snacks applies until 30 minutes after the last bell. After that, athletics events, theater, and community events can use an expanded planogram — but only if the contract addendum permits and the wellness committee approves.

Event-mode planogram
Operator timestamps planogram swap from compliant to event-mode after the last bell.
Event SKU set
Hydration + protein + concession-style snacks per athletics coordinator input.
Event restock
Operator pre-stocks before known events (home games, theater openings).
Event revenue routing
Event-mode revenue commission may differ from school-day rate; document in contract addendum.
Wellness alignment
Wellness committee approves event-mode SKU set annually.

5. Telemetry + reporting

Telemetry baselines the program. Per-SKU velocity + cashless mix + refund rate + uptime are the four KPIs reported monthly.

  • Cellular telemetry on every machine, sub-15-minute heartbeat (Cantaloupe / Nayax / 365 Retail Markets).
  • Per-SKU velocity report by location and by school-day vs after-school window.
  • Cashless mix report — target ≥ 75% at HS; below 60% triggers payment hardware review.
  • Refund rate report — target < 2% of transactions.
  • Uptime report — target ≥ 98% per machine per month.
  • Temperature excursion log for refrigerated units.
  • Smart Snacks attestation refreshed quarterly.

6. Revenue model + commission structure

Commission structures range from 10-30% of gross sales depending on operator service level, restock cadence, and equipment investment. Revenue routes per district policy — athletics, student activities fund, PTA partnership, or general fund.

7. Wellness committee + parent communication

  • Wellness committee reviews SKU list each quarter.
  • Parent communication via district channels (ParentSquare, Remind, district email) at launch and annually.
  • Parent feedback channel — annual survey or refund-form data review.
  • After-school SKU set posted on athletics + activities pages.
  • Annual KPI report shared with PTA + booster club.

8. Operator selection checklist

  • Operator has 24+ months of HS placements with 3 verifiable references.
  • Operator can attest to Smart Snacks workflow and after-school event mode.
  • Operator's telemetry and cashless capabilities meet contract spec.
  • Operator carries insurance (GL ≥ $2M / $5M aggregate, auto, workers' comp, cyber).
  • Operator's restock cadence is contractually flexible.
  • Operator reports monthly KPIs and quarterly Smart Snacks attestation.
  • Operator has executed a wellness-policy review with the school nutrition director.
This template is a working program-design reference. Localize to district wellness policy and state competitive-food rules; confirm board approval workflow before commit.
