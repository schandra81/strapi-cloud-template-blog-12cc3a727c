Vending Machine Repair Playbook

Parts inventory + preventive maintenance schedule + dispatch protocol

98%+ machine uptime is the difference between a vending business that compounds and one that bleeds anchor accounts. This playbook codifies the parts inventory, preventive maintenance schedule, and dispatch protocol the LetUsVending operator network uses — referenced against NAMA technician training, NEC 210.8 GFCI / 250.118 grounding, ASHRAE 15 refrigerant safety, EPA Section 608 refrigerant handling, FDA Food Code 2022 cold-chain (≤41°F), and OSHA 1910.147 lockout/tagout.

2. Preventive maintenance schedule

  1. Every visit (1-3x/week)

    Wipe glass + selection area, test 3 random selections, verify card reader online (telemetry), check internal temperature on refrigerated units, top-off coin/bill mech, photograph planogram.

  2. Monthly

    Clean coin/bill mech path, vacuum condenser coil (refrigerated units), check door gasket, test refund button, log refund rate, calibrate temperature sensor.

  3. Quarterly

    Lockout/tagout per OSHA 1910.147, deep-clean evaporator, inspect compressor for oil leaks, verify door switch + interlock, replace any worn coils, check belts/clutches in spiral machines, validate keypad responsiveness, full sanitize of cold-food units per FDA Food Code 2022.

  4. Annually

    Refrigerant pressure check (EPA Section 608 certified tech), MDB harness inspection, validator firmware update, card reader firmware update, OEM PM kit installation, scale removal on coffee/hot-beverage units.

  5. Every 5-7 yr / EOL

    Compressor + condensing-unit swap or whole-machine refresh decision; refurb cycle vs. replace decision per the operator's capex policy.

4. Telemetry → dispatch automation

Heartbeat
Modem heartbeat every 5-15 minutes (Cantaloupe Seed, Nayax, 365 Retail Markets, Vendsys). 30-minute offline triggers Tier-1 alert.
Cold-chain alert
Temperature > 41°F for > 30 minutes triggers Tier-4 alert. FDA Food Code 2022 §3-501.16 — corrective action required.
Payment failure alert
Card-reader decline rate >5% / hour triggers Tier-1 alert. Usually a card-reader firmware reset or RMA.
Stockout alert
SKU below threshold triggers Tier-2 — auto-attached to the next planned visit if telemetry sees adequate stock on adjacent SKUs.
Vibration / door alert
Suspected tilt / pry events trigger an immediate operations alert + camera-frame retrieval where the site supports it.

5. Common failure modes + first-pass fix

  1. Coil not vending (spiral machine)

    First-pass — check coil motor connector + voltage; replace coil motor; reseat MDB. 80% of cases. If persists, suspect main board.

  2. Cold-zone temperature alert

    First-pass — check condenser coil for dust, evaporator fan motor running, door gasket seal, ambient temperature. If sealed system, EPA Section 608-certified tech only.

  3. Bill acceptor jam / reject

    First-pass — clean validator path with cleaning card; firmware reset; if persists, swap to refurb unit, send original to RMA.

  4. Card reader 'no signal'

    First-pass — power-cycle modem (60 sec off), reseat SIM, check carrier coverage; if persists, swap unit; coordinate with the telemetry vendor on RMA.

  5. Spiral over-vend / under-vend

    First-pass — calibrate spiral home position, verify SKU sits flush, replace coil if worn.

6. Safety + compliance

  • OSHA 1910.147 lockout/tagout on any service that exposes a tech to electrical or mechanical hazard
  • EPA Section 608 certification for any refrigerant handling — universal or Type 1/2/3 as appropriate
  • NEC 210.8 GFCI receptacle within 6 feet of every machine on first install
  • NEC 250.118 grounding continuity check on every PM visit
  • Ladder + lift safety per OSHA 1910.23 / ANSI A14 if servicing overhead
  • Bloodborne pathogen kit on the truck for cleanup after vandalism / injury
  • Service-call log retained 3-5 years for AHJ + insurer reference

7. Refurb vs. replace decision

Refurb threshold
Replace compressor + condensing unit on a 8-12 year machine if frame, payment, and electronics are sound. ~$800-$2,200 vs. $4,500-$9,000 for new.
Replace threshold
Card-reader/electronics generation gap, frame corrosion, or third compressor failure in 5 years — replace.
Refresh cadence
Anchor accounts get refreshed equipment every 5-7 years — line item in the contract is the cleanest way to budget it.
