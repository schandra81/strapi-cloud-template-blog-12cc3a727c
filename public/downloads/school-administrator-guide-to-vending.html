School Administrator Vending Pack

Procurement framework, compliance overlay tracker, contract structure template, category review cadence

K-12 vending is one of the few campus services governed simultaneously by federal nutrition law (7 CFR 210.11), state competitive-food rules, district wellness policy, and standard procurement law. This pack codifies the four artifacts an administrator needs — procurement framework, compliance overlay tracker, contract structure template, and category review cadence — so the administrator is equipped to run an audit-ready program rather than improvise.

2. Compliance overlay tracker

Vending compliance has four layers. The tracker lets the administrator confirm operator attestation against each layer before commit and at annual recertification.

  1. Federal — Smart Snacks in School (7 CFR 210.11)

    All student-accessible SKUs during the school day pass the USDA Smart Snacks Product Calculator. Operator provides per-SKU attestation quarterly.

  2. Federal — FALCPA + FDA Food Code

    Allergen labels per FALCPA on every SKU; refrigerated units hold 35-40°F per FDA Food Code 3-501.16; HACCP-style temperature log for fresh-food.

  3. Federal — ADA Title III + Section 308

    Operable parts ≤ 48 inches above floor, ≥ 15 inches; ≤ 5 lbf operating force; clear floor space 30×48 inches; cashless option for users without coin/bill handling.

  4. State — Competitive food rules

    State agency rules layer above the federal floor; the school nutrition director confirms which state overlay applies.

  5. Local — District wellness policy (HHFKA § 204)

    Wellness committee approves SKU list, planogram, and after-school event mode.

  6. Local — Board policy

    Board approves vending contract and any revenue-sharing arrangement with athletics or PTA.

3. Contract structure template

The contract has six standard sections. Use this template as a starting point; legal counsel localizes for state law.

Scope
Number and type of machines, placement, audience, restock cadence, after-school event mode, donation pathway, and reporting.
Commission structure
Percentage of gross sales or fixed per-machine fee; payment frequency (monthly typical); cash and cashless commission rates can differ.
SLA tiers
Tier-1 (offline / payment / cold-chain): < 24-hour resolution. Tier-2 (stockout / planogram): 24-48-hour ack, 5-7-day resolution. Tier-3 (cosmetic): 14-day resolution.
Compliance attestation
Operator attests to Smart Snacks, FALCPA, FDA Food Code, ADA Section 308, and state competitive-food rules at install and quarterly.
Insurance + indemnification
GL ≥ $2M/$5M aggregate, auto, workers' comp, cyber. Operator indemnifies district for product liability per state cap.
Term + termination
Initial term 36-60 months; termination for cause (SLA breach, compliance failure) on 30 days' notice; termination for convenience on 90 days' notice.

6. Annual recertification checklist

  • Smart Snacks attestation refreshed for full SKU list.
  • ADA Section 308 site visit completed; measurements logged with photo.
  • FDA Food Code temperature logs reviewed; no excursions outstanding.
  • Insurance certificates current (GL + auto + workers' comp + cyber).
  • Wellness committee minutes attached to procurement file.
  • Annual KPI trend report attached (12-month stockout, cashless mix, refund rate).
  • State competitive-food overlay reviewed against any state-rule changes.
  • Board re-consent if contract term renewal applies.

7. Common administrator pitfalls

These are the six issues that most often surface in K-12 vending audits. Document mitigations in the operator contract.

Smart Snacks drift
SKU added by operator without per-SKU attestation. Mitigation: quarterly attestation + sample audit.
After-school window confusion
Operator continues compliant planogram after last bell, losing event-mode revenue. Mitigation: contract addendum defines event-mode SKUs.
Temperature excursion gap
Refrigerated unit deviates outside 35-40°F without alarm. Mitigation: 4-hour SLA alarm + monthly review.
Cashless gap
Cashless coverage < 60% leaves free/reduced-eligible students without access. Mitigation: student-ID integration.
Refund opacity
Refund rate not reported; member complaints accumulate. Mitigation: monthly KPI report + quarterly review.
Wellness committee disengagement
Committee skips quarterly review. Mitigation: standing agenda item + 7-day pre-meeting packet.
