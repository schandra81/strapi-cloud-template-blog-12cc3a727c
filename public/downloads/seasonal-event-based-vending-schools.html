Seasonal School Vending Playbook

Seven planning cycles, academic calendar coordination, event-day surge protocols

School vending demand is not uniform — it spikes for back-to-school, fall sports, semester finals, winter performance season, state testing, spring sports, and graduation. This playbook codifies the seven planning cycles, the academic calendar coordination protocols, and the event-day surge protocols that keep machines stocked during peak and lean when traffic disappears.

2. Academic calendar coordination

Each district publishes an academic calendar with school days, holidays, parent-teacher conferences, testing dates, and event dates. Operator integrates this calendar into route planning so restock cadence flexes with traffic.

  1. Calendar import

    District provides academic calendar (ICS or CSV) at start of school year; operator imports into route-planning system (Vendsoft, VendingMetrics, VendMAX) for the year.

  2. Surge calendar overlay

    Athletic + activity + testing schedules layered onto academic calendar; restock cadence flexes per placement per event.

  3. Closure flagging

    Holidays + breaks + early-dismissal days flagged; operator suspends restock + dispatches PM visits during longer closures.

  4. Calendar change notification

    District notifies operator of calendar changes (weather closures, schedule changes) by 6 AM same-day; operator adjusts route.

3. Event-day surge protocols

Home games, tournaments, graduation, prom, and performances drive 3-5x normal traffic at the corresponding placement. Surge protocol pre-positions stock + adds same-day restock.

  • Game-day gym placement — pre-stock day-before; same-day mid-game top-off; expand electrolyte + sports nutrition + grab-and-go
  • Tournament weekend — daily restock through tournament; portable backup machine if traffic 2x+ normal capacity
  • State / regional testing week — testing-morning restock; expand protein bars + low-sugar + water at testing-room-adjacent placements
  • Performance night (band, drama) — afternoon restock for evening crowd; expand mainstream snack + beverage at performance-venue placements
  • Graduation week — daily restock; portable backup at outdoor ceremony venue with shaded power source
  • Prom / dance — evening top-off; expand mainstream + hydration at venue-adjacent placement
  • Welcome week / orientation — daily restock; planogram refresh for new students

5. Smart Snacks compliance — seasonal overlay

Seasonal planogram changes must continue to pass 7 CFR §210.11 during school day. Operator + district nutrition services review every seasonal planogram before launch.

  • Pre-season planogram submitted to district nutrition services 2 weeks before launch
  • Each new SKU has Smart Snacks Product Calculator result + USDA-aligned vendor attestation on file
  • After-school + event-day planogram (when Smart Snacks doesn't apply) clearly distinguished in plan-of-record
  • Faculty-only placement planogram (Smart Snacks doesn't apply) clearly distinguished
  • Quarterly compliance audit covers every seasonal SKU rotated through the prior quarter

6. Operational support structure

Restock cadence — peak weeks
4-5 visits/week at high-traffic placements; daily for event-day surge.
Restock cadence — lean weeks
0-1 visit/week during winter / summer break; PM visits scheduled during these windows.
Service SLA — Tier 1
Offline, cold-chain, payment — <24 hr ack + <24 hr resolution.
Event-day SLA add-on
Same-day top-off pre-game / pre-event; emergency restock <4 hr if portable backup needed.
Refund flow
Refund via app, on-machine touchpad, or front-office. Quarterly refund report; event-day refund spike investigated.
