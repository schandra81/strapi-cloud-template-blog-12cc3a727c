Perishable Vending Program Design Template

Cold-chain SOP, equipment specification, restock cadence for refrigerated and hot-hold vending

Temperature-controlled vending for perishable SKUs is governed by FDA Food Code 3-501.16 (cold-hold ≤ 41°F, hot-hold ≥ 135°F), HACCP-style temperature logging, FALCPA allergen labeling, ASHRAE Standard 24 refrigerated transport, and operator SLA on excursion response. This template codifies three artifacts — cold-chain SOP, equipment specification, restock cadence — so a facility is equipped to commit a compliant perishable program rather than fail at the first temperature excursion.

1. Cold-chain SOP

The SOP defines temperature targets, monitoring cadence, excursion response, and recordkeeping. Operator attests to each at install and at quarterly review.

  1. Temperature targets

    Cold-hold ≤ 41°F (FDA Food Code 3-501.16); target operating range 35-40°F to provide margin. Hot-hold ≥ 135°F.

  2. Monitoring cadence

    Telemetry reports internal temp every 15 minutes; daily summary to operator dashboard. Calibrated thermometer onsite for spot verification.

  3. Excursion alarm

    4-hour SLA alarm at any deviation outside 35-40°F (cold) or below 135°F (hot). Auto-alert to operator + host EHS.

  4. Excursion response

    If excursion > 4 hours: discard all temperature-sensitive SKUs per FDA Food Code 3-501.19 time-as-public-health-control rules; log discard volume.

  5. Recordkeeping

    Temperature log retained 12 months minimum; HACCP-style critical control point records per FDA Food Code 8-203.13.

  6. Maintenance cadence

    Quarterly compressor + coil inspection; annual refrigerant + door-seal inspection; documented in operator service log.

3. Restock cadence framework

Restock cadence is tighter for perishables than for ambient SKUs. Cadence is driven by per-SKU shelf life and per-SKU velocity.

Fresh-food entrée / salad / sandwich
Daily restock; pull at 2 days from best-by date.
Yogurt / cheese / cold dairy
2-3x/week; pull at 3 days from best-by.
Refrigerated beverages (milk, juice)
1-2x/week; pull at 7 days from best-by.
Hot-meal entrée
Daily restock; pull at end of each shift cycle.
Hot-drink (coffee, soup)
1-2x/week; refresh hot-hold daily.

4. FALCPA allergen labeling

  • Big-9 allergens (milk, eggs, fish, shellfish, tree nuts, peanuts, wheat, soy, sesame) declared in plain language on each SKU.
  • Allergen summary on machine face visible from clear floor space (ADA Section 305).
  • Operator attests to manufacturing-site cross-contact disclosures.
  • Allergen UI flag in machine selection per ADA Section 707.
  • Allergen incident protocol documented; school nurse / EHS coordinator on incident escalation list.

6. Operator capability checklist

  • Operator has 24+ months of cold-chain placements with documented FDA Food Code attestation.
  • Operator's telemetry reports 15-min internal temperature on every refrigerated unit.
  • Operator's 4-hour SLA alarm tested and documented.
  • Operator's restock crew trained on FIFO + 2-day pull rule.
  • Operator attests to FALCPA allergen labeling + cross-contact disclosures.
  • Operator's equipment uses AIM Act compliant refrigerant (GWP ≤ 700).
  • Operator's HACCP-style records retained 12 months minimum.
  • Operator carries insurance covering product liability + cold-chain failure.

7. QBR review framework

  • Temperature excursion log review — count + duration + corrective action.
  • Discard volume report — per-SKU volume + root cause.
  • Restock cadence adherence — actual vs SLA cadence.
  • Per-SKU velocity report — Tier-1/2/3 movements.
  • FALCPA allergen attestation refresh.
  • Refrigerant inventory + AIM Act compliance refresh.
  • Action items + owner + due date.

8. Annual recertification

  • FDA Food Code 3-501.16 attestation refreshed.
  • Telemetry firmware updated to current stable version.
  • Calibrated thermometer NIST-traceable verification onsite.
  • Refrigerant inventory + leak-check per EPA Section 608 (applicable units).
  • Door-seal inspection + replacement where needed.
  • HACCP-style record audit — 12-month log reviewed for completeness.
  • Allergen + FALCPA attestation refreshed for full SKU list.
This template is a working cold-chain reference. Confirm FDA Food Code state adoption (most states adopt with minor amendments) with the facility EHS coordinator and operator before commit.
