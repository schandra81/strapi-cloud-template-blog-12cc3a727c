Manufacturing Vending Setup Pack

RFP criteria, site survey checklist, contract framework, and transition playbook

Setting up vending in a manufacturing plant the right way takes 60–90 days from kickoff to go-live. This pack consolidates the four working documents a facility services + HR team needs to run the procurement: RFP criteria with scoring weights, site survey checklist, contract framework, and a transition playbook for new program or operator change.

2. Site survey checklist

  • Workforce headcount per shift; shift hours; rotating Saturday or 24/7 coverage flag
  • Break-room footprint and pedometer flow at shift-change peak
  • Existing vending or food-service equipment + condition
  • Dedicated 20A circuits available or required new install (estimate $400–$1,500 each)
  • Cellular signal strength: Verizon / AT&T / T-Mobile RSRP measurement at each placement; target ≥ −90 dBm
  • Network — facility VLAN drop available with PCI segmentation attestation, or cellular-only preferred
  • Egress + NFPA 101 §7.1 corridor width at each placement (≥ 44 in)
  • Sprinkler clearance ≥ 18 in below deflector at each placement (NFPA 13 §8.6)
  • Forklift / MHE path offset; OSHA 29 CFR 1910.176(a) aisle clearance
  • ADA reach range, clear floor space, operating force feasibility at each placement (§§308, 305, 309)
  • PPE zone overlay — placement strictly in break room, never on active production floor
  • Climate control at placement zones (ambient 50–90°F for refrigerated cabinets per NSF-7 spec)
  • Photographs of each placement at install + measurements retained 5+ years

3. Contract framework — key clauses

  1. Term + termination

    Typical 3-year initial term with 1-year auto-renew. Termination for convenience with 60–90 day notice; termination for cause with 30-day cure.

  2. SLA + service credit

    Three-tier SLA with remedies. Service credit = 50% of 90-day service fee if Tier 1 missed > 2x in 90-day period OR uptime < 95% for 2 consecutive months.

  3. Commission + subsidy mechanics

    Commission % tiered by site revenue (0% under threshold; 5–25% above). Subsidy mechanic via payroll-deduct or wellness account; reconciliation monthly.

  4. Audit rights

    Host has audit right on 30-day notice; operator opens commission books for inspection. Quarterly reports include unit counts + revenue per machine.

  5. Insurance + indemnity

    $1M / $2M GL + product liability + cyber if cashless. Cross-indemnity for third-party claims; mutual hold-harmless.

  6. PCI + data

    Operator attests to PCI-DSS v4.0 AoC current. Employee data (badge ID) handled under data terms; no PHI; no patient or customer data.

  7. ADA + safety attestation

    Operator attests in writing at install + annually to §§308, 305, 309 compliance; recertification photos + measurements.

  8. Transition + end-of-term

    Equipment removal at operator cost. Customer data + planogram history portable; operator returns or destroys per host instruction.

4. Transition playbook — new program

  1. Week 1–2: kickoff + planogram principles

    Joint kickoff with operator AM + facility services + HR + wellness committee. Planogram principles drafted (AHA-aligned %, dietary inclusivity, subsidy mechanics).

  2. Week 3–4: install prep

    Electrical install scheduled; cellular survey completed; planogram loaded; signage + comms drafted.

  3. Week 5–7: install + integration

    Equipment installed; telemetry verified; badge + payroll-deduct integration tested in staging; soft-launch shift 1.

  4. Week 8: workforce onboarding

    Onboarding comms 2 weeks pre-launch; signage at break-room entry; QR-coded refund flow; lunch-and-learns for committee members.

  5. Week 9–10: full go-live + first telemetry cycle

    All shifts live; first per-shift velocity report at end of week 10.

  6. Week 11–13: 90-day review

    Quarterly business review with operator. Decide expansion, contraction, or hold. Document lessons learned.

5. Transition playbook — operator change (recompete)

Operator change mid-contract or at end-of-term has its own playbook. Below covers the typical 60-day window.

  1. T-60 days: notice + parallel planning

    Notice to outgoing operator. New operator runs site survey + signs contract. Workforce comms drafted.

  2. T-30 days: hardware swap planning

    Confirm whether equipment is operator-owned (most common in full-service) — outgoing operator removes; new operator installs. Refund tail handled by outgoing.

  3. T-14 days: workforce comms

    All-hands comms; signage refresh; new app / SMS / refund flow communicated.

  4. T-0: cutover

    Outgoing equipment removed in 24–48 hours; new equipment installed within 24–72 hours; signage updated; planogram loaded.

  5. T+30 days: stabilization review

    First-month telemetry; refund + escalation pattern review; any open issues from transition closed.

7. Common setup mistakes — to avoid

  • Skipping cellular signal verification → telemetry SLA fails at fringe-signal placements
  • Skipping ADA measurements → § 308/305/309 failure becomes litigation exposure
  • Subsidy mechanic via discount-at-PoS instead of badge ID → no per-FTE participation visibility
  • Workforce comms skipped or rushed → low participation; subsidy budget wasted
  • Contract missing service-credit remedy → no leverage when SLA missed
  • No quarterly business review cadence → planogram drift; no continuous improvement
  • Operator without PCI-DSS v4.0 AoC selected → cyber liability falls on facility
  • No fresh / refrigerated capability → real-meal share stays low; participation plateaus
