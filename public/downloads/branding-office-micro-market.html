Office Micro-Market Branding Playbook

Palette, signage, kiosk skin, wall graphics, accessibility, and workplace navigation integration

A branded micro-market reads as part of the workplace — not a vendor installation. This playbook codifies palette, signage, kiosk skin, wall graphics, and accessibility decisions against ADA Title III 2010 Standards, WCAG 2.2 AA, ANSI Z535.1 safety color, and ISO 7010 wayfinding so design choices clear procurement and facilities review the first time.

1. Brand-system inputs

Start with the corporate brand book. Pull the items below before any visual design begins so the operator's design partner is briefed correctly.

Color tokens
Primary, secondary, neutral, accent. Provide hex + Pantone Coated + CMYK + RAL for printed and powder-coated surfaces.
Typography
Display + body type families with licensing notes. Provide TTF/OTF files for kiosk-screen overlay and signage.
Logo lockups
Horizontal, stacked, mark-only. Minimum clear space and minimum size for kiosk vinyl wrap and overhead signage.
Tone-of-voice
Copy guidance — formal/conversational, sentence case vs. title case, voice samples for shelf signage.
Photography style
Studio-on-white vs. lifestyle, lighting temperature, do-not-use shots.

3. Signage hierarchy

  1. Identity sign (overhead)

    Cabinet name + employer brand mark. 4-6 inch cap height for visibility from 15-30 feet. Mounted with workplace wayfinding (ISO 7010 family) so signage 'reads as the building.'

  2. Category signage (shelf strip)

    Snacks / Drinks / Fresh / Coffee / Healthy-Share. 1-1.5 inch cap height, sentence case, sans-serif. Color-coded but always with text label (WCAG 2.2 1.4.1 — color not the only signal).

  3. SKU labels (planogram)

    Product name, price, key allergens, calories. FALCPA-compliant allergen statement on every SKU. Calorie disclosure on chains of 20+ locations per 21 CFR 101.11.

  4. Kiosk-screen overlay

    Welcome screen with employer logo and tagline. WCAG 2.2 AA color contrast (4.5:1 text, 3:1 large text / UI). Skin file delivered as a single PSD/AI per operator spec.

  5. Wayfinding signage

    Hallway directional arrows referencing the market by its branded name; coordinate with facilities-management signage program.

4. Kiosk skin file delivery checklist

  • Vector source (AI, EPS, or SVG) at 1:1 scale
  • 300 dpi raster fallback for photography or gradients
  • Fonts outlined or supplied as TTF/OTF with license confirmation
  • Pantone Coated + RAL match alongside hex
  • Bleed 0.125 in around vinyl wrap die-line
  • Touchscreen + reader die-lines preserved (no overlap with controls)
  • ADA reach-range overlay layer for kiosk-button labels (Section 308)
  • Two color treatments approved (light + dark) for daylight and night-shift legibility

5. Accessibility-by-design checks

Every brand decision is checked against ADA Title III + WCAG 2.2 AA + ANSI Z535 before sign-off — this prevents costly reprints after install.

Text contrast
≥4.5:1 normal text, ≥3:1 large text and UI components (WCAG 2.2 1.4.3 / 1.4.11).
Operable-part contrast
Buttons, readers, dispense flap visually distinct from cabinet (≥3:1 against background).
Font size minimums
Body text ≥ 16 px equivalent on kiosk screen; ≥ 5/8 in cap height for price displays per ADA 703.5.5 where required.
Tactile signage
Raised characters + Grade 2 Braille for any permanent room/space identification per ADA 703.2-703.4.
Audio + caption
Any video content captioned (WCAG 2.2 1.2.2) and includes audio description option (1.2.5).

6. Workplace-navigation integration

  1. Floor-plan registration

    Market name added to building directory, suite-level wayfinding, and digital signage layer (Appspace / Korbyt / etc.). Coordinate with facilities-management lead.

  2. Floor decals

    Approach decals 6-10 feet from entry, color-matched to wayfinding system. Anti-slip rating ≥ COF 0.5 per ANSI A326.3.

  3. QR code + intranet entry

    QR on kiosk + wall sign deep-links to intranet page with hours, contact, refund instructions, allergen filter, current planogram.
