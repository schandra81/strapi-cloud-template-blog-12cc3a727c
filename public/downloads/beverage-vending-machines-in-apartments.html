2. Planogram per zone

Per-zone SKU mix should reflect time-of-day pattern and resident behavior, not a single default planogram.

Lobby / mail-package
Bottled water (multiple sizes), still + sparkling, premium soda, kombucha, cold brew, iced tea, kids juice pouch. ~35-40 SKUs.
Fitness center
Sports drinks (Gatorade, BodyArmor), electrolyte (LMNT, Liquid IV), water, protein shakes (Premier, OWYN), coconut water, low-cal energy. ~20-25 SKUs.
Pool deck
Bottled water + sports drink + kid juice + canned wine seltzer (where age-gated). Compact 15-20 SKU outdoor cabinet.
Resident lounge / co-work
Premium coffee RTD (La Colombe, Stumptown), cold brew, sparkling water, kombucha, kids juice. ~25-30 SKUs.
Dog wash / pet area
Out of scope for beverage; place compact pet-treat / waste-bag machine instead.
Bike room / parking
Cold water + sports drink + canned-coffee compact stack cooler.

4. Member-app pricing tier

Residents using the building member-app see preferred pricing; visitors / guests see standard pricing. Standard on Nayax Onyx, Cantaloupe Engage, 365 Retail Markets.

Member tier
Base × 0.85-0.95 (10-15% resident discount funded by operator margin); cashless via member-app SDK or RFID badge.
Guest tier
Base × 1.0; cashless or cash via standard payment terminal.
Subsidized (host-paid)
Optional — owner credits per-month stipend per resident; residents see further-reduced or $0 price.
Authentication
Member-app credential, building-app SDK token, or RFID resident badge integrated with PMS (RealPage, Yardi, Entrata).

6. Service + restock framework

Restock cadence
Lobby + lounge — weekly; fitness + pool seasonal — 2x weekly Q2-Q3; corridor + bike — bi-weekly.
Telemetry SLA
Sub-15-minute heartbeat, alerts on offline + temperature excursion + payment failure.
Cold-chain SLA
Beverage cabinets internal temp 35-40°F; excursion > 30 min triggers 4-hour response.
Refund SLA
Cashless 24-hour auto-refund via member-app or QR; cash within 7 days on next restock.
Planogram refresh
Quarterly review — telemetry velocity report drives 15-25% SKU rotation per quarter.
